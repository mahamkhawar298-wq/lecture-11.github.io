# EduNexus 🏛️ | Virtual Classroom Management Suite

A comprehensive virtual classroom management system with student tracking, attendance, assignments, analytics, and real-time notifications.

## Overview

EduNexus is a full-featured classroom management platform with 8 core modules, data visualization, file attachments, PDF reports, and localStorage persistence — all in pure HTML/CSS/JS.

## Features

| Module | Features |
|--------|----------|
| 📊 Dashboard | Attendance chart, most absent students, pending assignments, announcements, class countdown |
| 👨‍🎓 Students | Search, filter, pagination, details modal, PDF report cards |
| 📅 Attendance | Weekly calendar, mark present/absent, week navigation |
| 📚 Subjects | Add/delete subjects, teacher directory, email tracking |
| 📝 Assignments | File attachments, student submissions, completion tracking |
| 📢 Announcements | Create posts, pin important, chronological order |
| 👤 Profile | Bio editing, attendance stats, submission count |
| 📈 Analytics | 7-day trend chart, export as PNG |
| 🔮 Random User | API integration, one-click add to class |

## System Features

- ✅ Undo/Redo (Ctrl+Z / Ctrl+Y) with 30-action stack
- ✅ Dark mode with persistence
- ✅ Real-time notifications
- ✅ JSON backup & restore
- ✅ Auto-save to localStorage
- ✅ Fully responsive design

## Tech Stack

HTML5 · CSS3 · JavaScript  · jsPDF · Font Awesome · RandomUser.me API
